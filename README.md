# 🌦️ Weatherly — Real-Time Global Weather Dashboard

> A modern real-time weather web application that delivers live weather updates, global search, and 36-hour forecasting with dynamic UI animations.

---

## 🚀 Live Overview

Weatherly is a fast, responsive weather dashboard that allows users to:
- Search any city, state, or country 🌍
- Get real-time weather instantly ⚡
- View Live Location Weather
- View next 36-hour detailed forecast ⏱️
- Experience dynamic animated themes based on weather 🎨

---

## ✨ Features

### 🌍 Global Weather Search
Search any location worldwide:
- Cities
- States
- Countries

Instant results with real-time API response.

---

### 📍 Live Location Detection
Automatically detects user location and displays current weather instantly.

---

### ⏱️ 36-Hour Forecast
Advanced forecasting system showing:
- Hour-by-hour temperature changes
- Weather condition updates
- Smooth forecast timeline visualization

---

### 🎨 Dynamic Weather Themes
UI changes based on weather conditions:
- ☀️ Sunny → Bright theme
- 🌧️ Rain → Dark rainy animation
- ⛈️ Thunderstorm → Storm effects
- ☁️ Cloudy → Soft neutral UI

---

### ⚡ Fast & Responsive
- Optimized API calls
- Lightweight frontend
- Works on mobile, tablet, and desktop

---

## 🧠 How It Works

1. User enters a location  
2. App sends request to OpenWeather API  
3. Weather data is fetched in real-time  
4. 36-hour forecast is processed  
5. UI updates dynamically with animations  

---

## 🛠️ Tech Stack
